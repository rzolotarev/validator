Данный код - проверка на ошибки заполненного бюллютеня для голосования. Бюллютень - список вопросов для голосования. В данном  слувчае 3 типа вопросов:
простой(Simple), кумулятивный(Qumulative) и раздельный (Separate) 
Стартовая точка - класс Validator:
1. В констукторе строится дерево на основании бюллютеня, вершинами которого являются правила, 
которые должны быть соблюдены (свойство IsFulfiled), чтобы бюллютень считался действительным, в случае не соблюдения хотя бы одного правила 
бюллютень признается недействительным и пользователь должен увидить причину (метод GetErrorText());
2. При построении дерева каждому правилу соотвествуют элементы интерефейса - класс DocField. 
Свойство DependsOn извлекает такие элементы. 
Свойство DependantRulesDependOn извлекает все элементы интерфейса, которые находятся под данным правилом;
3. На выходе имеем массив элементов интерфейса со статусом проверки и текстом, в случае существующих ошибок.
